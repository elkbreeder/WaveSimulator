Compute net updates for the cell on the left/right side of the edge. This is an expert method, because a lot of (numerical-\/)knowledge about the problem is assumed/has to be provided. It is the f-\/\+Wave entry point for the hybrid solver, which combines the \char`\"{}simple\char`\"{} F-\/\+Wave approach with the more complex Augmented Riemann \hyperlink{namespaceSolver}{Solver}.

wet\+Dry\+State is assumed to be Wet\+Wet.
